e# To-Do List App (Python Project)

## Project Overview

To-Do List App is a simple Python-based application that helps users organize and manage their daily tasks. The application provides options to add new tasks and view existing tasks through a command-line interface. This project demonstrates the use of lists, loops, conditional statements, and user input handling in Python.

## Objective

The goal of this project is to help users keep track of their daily activities by maintaining a simple task list.

## Features

* Add new tasks
* View all saved tasks
* User-friendly command-line interface
* Easy task management
* Real-time task storage during program execution

## Technologies Used

* Python 3

## Concepts Applied

* Variables
* Lists
* User Input
* While Loops
* Conditional Statements
* List Operations
* Enumeration (`enumerate()`)

## How It Works

1. The program displays a menu with available options.
2. Users can add tasks to the task list.
3. Users can view all saved tasks.
4. The program continues running until the user chooses the Exit option.
5. Tasks are stored in a Python list during execution.
