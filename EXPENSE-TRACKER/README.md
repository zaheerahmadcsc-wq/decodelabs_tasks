# Expense Tracker (Python Project)

## Project Overview

Expense Tracker is a simple Python-based application that allows users to record their expenses and calculate the total amount spent. This project demonstrates the use of variables, loops, conditional statements, user input handling, and accumulator logic in Python.

## Objective

The goal of this project is to help users keep track of their daily expenses by entering expense amounts and calculating the total expenditure.

## Features

* Add multiple expense amounts
* Automatically calculate total spending
* User-friendly command-line interface
* Simple and easy-to-understand code
* Demonstrates accumulator pattern for data processing

## Technologies Used

* Python 3

## Concepts Applied

* Variables
* User Input
* While Loops
* Conditional Statements
* Type Conversion
* Mathematical Operations
* Accumulator Logic

## How It Works

1. The user enters expense amounts.
2. The program continuously accepts expenses until the user types `done`.
3. Each expense is added to the running total.
4. The program displays the total amount spent.
